📌 Project Overview

The HR Analytical Dashboard is an interactive data visualization project built using Tableau to analyze workforce data and generate actionable HR insights. This dashboard enables organizations to monitor employee trends, track attrition, and improve decision-making through data-driven strategies.

🎯 Objectives
Analyze employee demographics and workforce distribution
Monitor attrition rates and identify key factors
Evaluate department-wise performance
Provide insights to improve employee retention and HR planning
🛠️ Tools & Technologies
Tableau Desktop
Data Visualization & Dashboard Design
Data Cleaning & Preparation
Calculated Fields & Parameters
Interactive Filters & Actions
📂 Dataset Description

The dataset contains HR-related data, including:

Employee ID
Age, Gender, Education
Department & Job Role
Salary / Income Details
Attrition Status
Years of Experience

🔹 Interactive Dashboard
Dynamic filters for customized analysis
User-friendly and visually appealing design
🔹 Attrition Analysis
Attrition rate overview

Key factors influencing employee turnover
🔹 Workforce Insights
Employee distribution by department and role
Gender diversity analysis

🔹 Salary & Experience Analysis
Salary distribution across roles
Experience vs attrition trends
📈 Key Insights
Identifies departments with high employee turnover
Highlights salary gaps and workforce imbalance
Provides demographic insights for diversity improvement
